# Jupyter Lab Notes on SHACL

## Installation

```
git clone https://github.com/datadavev/shacl-pad.git
cd shacl-pad
python -m venv env
source env/bin/activate
pip install -U pip
pip install -r requirements.txt
```

## Running the notebooks

In the `shacl-pad` folder:

```
jupyter lab
```
